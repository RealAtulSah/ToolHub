# Master Prompt - ToolHub Pro

## Objective

Create a complete production-ready All-in-One Tools Website named ToolHub Pro using HTML5, CSS3 and Vanilla JavaScript.

The website must be modern, responsive, lightweight and SEO friendly.

No backend should be used.

All tools must work entirely inside the browser.

Use open-source JavaScript libraries wherever necessary.

The code should be modular, clean and separated into folders.
